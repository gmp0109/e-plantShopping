# Paradise Nursery

## Project Description

Paradise Nursery is a React-based e-commerce web application that allows users to browse and purchase a variety of indoor plants. The application provides plant categories, product details, shopping cart functionality, and quantity management using Redux for state management.

## Features

* Browse plants by category
* View plant images, descriptions, and prices
* Add plants to the shopping cart
* Update product quantities
* Remove products from the cart
* View total cart amount
* Responsive user interface built with React

## Technologies Used

* React
* Redux Toolkit
* React Redux
* Vite
* GitHub Pages

## Purpose

The purpose of this project is to demonstrate the use of React components, state management with Redux Toolkit, and deployment using GitHub Pages.
